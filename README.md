# Employee Database Deduplication

## Overview

This project is developed as part of the M-Tech Data Science Internship.

The Employee Database Deduplication System is designed to clean HR employee records by identifying and removing duplicate entries, standardizing employee names, and generating a clean dataset. The project also includes a PyQt5 graphical user interface (GUI) for easy interaction.

## Features

- Load employee dataset from Excel
- Detect duplicate employee records
- Remove duplicate entries
- Standardize employee names
- Display data in a GUI table
- Save cleaned dataset to Excel
- Basic Exploratory Data Analysis (EDA)
- Data visualization using Matplotlib

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- PyQt5
- OpenPyXL
- Jupyter Notebook
